# 物料导入导出说明

## 功能目标

物料主数据可以通过 CSV 批量导入和导出，减少手工逐条录入。

当前只处理物料主数据，不通过导入文件直接改变库存数量、库存金额和成本层。库存初始化、盘盈、盘亏、误差抹平仍然应在库存管理中走存货调整。

## 字段口径

| 字段 | 说明 |
| --- | --- |
| 物料编码 | 业务可视化编码，必须填写且唯一。可以使用客户零件号。 |
| 物料名称 | 人工识别用名称，不建议全部写成同一个泛称。 |
| 规格 | 尺寸、材质、颜色、图纸版本等技术描述。 |
| 备注 | 额外说明，例如客户零件号说明、图纸版本、特殊检验要求。 |
| 分类 | 支持 `RAW`、`FINISHED`、`AUXILIARY`、`SCRAP`、`DEFECTIVE`、`PACKAGING`、`OTHER`，也支持常用中文分类。 |
| 客户编码 | 可选。填写时必须已经存在于客户基础资料中。 |
| 库存单位 | 必填，例如件、根、米、kg。 |
| 启用双单位 | 可填是/否。启用时必须填写核算单位和换算系数。 |
| 核算单位 | 成本核算单位，例如 kg。未启用双单位时默认等于库存单位。 |
| 换算系数 | 1 个库存单位对应多少核算单位。 |
| 成本方法 | 支持 `WEIGHTED_AVERAGE` 和 `FIFO`，也支持中文“移动加权平均”“先入先出”。 |
| 换算说明 | 可选，用于说明理论重量、实际称重等来源。 |

## 操作入口

- 物料管理顶部按钮：`导入`
- 物料管理顶部按钮：`导出`
- 导入弹窗内：`下载导入模板`
- 物料管理顶部筛选区：`显示字段`

## 显示字段控制

物料列表支持按浏览器保存显示字段偏好。名称和操作列固定显示，其他字段可以按需勾选。

可控制字段包括：图片、编码、分类、客户、规格、备注、库存单位、核算单位、库存、核算库存、创建时间。

该设置只影响当前浏览器的列表和卡片显示，不改变数据库内容，也不影响导入导出。

## 分页显示

物料管理默认按 20 条/页加载，支持切换为 50、100、200 条/页。筛选条件、客户筛选、分类筛选、每页数量变化后会回到第一页，避免因为当前页超出结果范围导致误以为数据丢失。

## 导入模式

| 模式 | 行为 |
| --- | --- |
| 跳过已有物料 | 默认模式。CSV 中编码已存在时不覆盖资料。 |
| 更新已有物料资料 | 覆盖名称、规格、备注、分类、客户、单位、换算关系和成本方法，但不修改库存余额。 |

## 校验规则

- CSV 必须包含表头：`物料编码`、`物料名称`、`库存单位`。
- 文件内物料编码不能重复。
- 物料编码不能占用已归档物料的编码。
- 客户编码如果填写，必须对应未归档客户。
- 启用双单位时，核算单位不能为空，换算系数必须大于 0。
- 任意一行校验失败时，整次导入不写入数据库。

## 验收清单

1. 下载模板，确认 CSV 能被表格软件打开。
2. 使用模板新增一条新物料，导入后物料列表能看到该物料。
3. 确认新物料在库存管理中有对应库存行，初始库存为 0。
4. 再次导入同一文件并选择“跳过已有物料”，应显示跳过数量。
5. 修改名称后选择“更新已有物料资料”，应能覆盖物料资料，库存数量不变。
6. 填写不存在的客户编码，导入应失败并显示具体行号。
7. 导出当前筛选结果，确认 CSV 内容与页面筛选一致。
8. 在显示字段中取消勾选“编码”和“备注”，确认列表/卡片不显示这些字段；刷新浏览器后偏好仍然保留。
9. 创建超过一页的物料，确认分页可以切换上一页、下一页、首页、末页，且每页数量切换后显示范围正确。
