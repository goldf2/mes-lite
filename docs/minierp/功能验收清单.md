# MiniERP 功能验收清单

用于人工核查当前系统的功能覆盖、数据逻辑和状态流转。

## 验收前提

- 本地服务运行在 `http://localhost:3002`
- 已执行 `npm run build`
- 使用管理员账号 `admin / admin123`
- 所有测试数据统一使用 `TST-FULL-*` 前缀，验收后清理

## 功能列表与检查标准

### 1. 登录与认证

- 测试方式：未登录访问受保护接口、错误密码登录、管理员登录、读取当前用户。
- 检查标准：未登录返回 `401/403`；错误密码返回 `401`；管理员登录成功且返回 `ADMIN`。

### 2. 权限矩阵

- 测试方式：管理员读取权限矩阵。
- 检查标准：返回角色、功能页、权限动作，且包含 `materials`、`orders`、`stats` 等资源。

### 3. 注册与审核

- 测试方式：注册普通操作员、未审核登录、管理员审核、审核后登录。
- 检查标准：新账号状态为 `PENDING`；未审核账号登录被拒绝；审核后状态为 `ACTIVE` 且可登录。

### 4. 权限限制

- 测试方式：普通操作员尝试新增物料。
- 检查标准：默认无权限时返回 `403`。

### 5. 物料管理

- 测试方式：新增、修改、查询、归档物料、在归档记录中恢复。
- 检查标准：新增 `201`；修改后字段正确；归档后列表不可见；恢复后重新可见；重复编码被拦截。

### 6. 物料图片

- 测试方式：上传多张图片、切换封面、查看列表主图、文件预览、归档封面。
- 检查标准：第一张自动成为封面；切换后仅一个封面；列表主图与封面一致；预览返回图片；归档封面后自动回退。

### 7. 供应商管理

- 测试方式：新增、修改、归档供应商、在归档记录中恢复。
- 检查标准：新增与修改成功；归档后不可再用于创建新来料单；恢复后重新可用。

### 8. 来料管理

- 测试方式：创建来料单、上传原始单据、确认收货、拒收。
- 检查标准：金额等于 `qty * unitPrice`；附件可上传；收货后库存增加；拒收后状态为 `REJECTED`。

### 9. 生产基础数据

- 测试方式：创建测试产品、BOM、默认工艺路线。
- 检查标准：产品可在列表中读取；BOM 和工艺路线可用于创建工单。

### 10. 工单管理

- 测试方式：创建工单、确认工单、领料、重复领料拦截。
- 检查标准：创建后物料完成预留；确认后状态为 `CONFIRMED`；领料后状态为 `PICKED`；重复领料被拒绝。

### 11. 库存账

- 测试方式：检查工单创建前后、领料后的库存变化。
- 检查标准：创建工单后 `qty=100 reserved=20 available=80`；领料后 `qty=80 reserved=0 available=80`。

### 12. 派工管理

- 测试方式：创建派工、确认派工、开工、完工。
- 检查标准：状态依次流转，最终为 `COMPLETED`。

### 13. 报工与质检

- 测试方式：首道报工、末道报工、创建质检记录、成品入库。
- 检查标准：首道报工后工单进入 `RUNNING`；末道报工后进入 `QC_WAITING`；质检通过后可入库。

### 14. 发货管理

- 测试方式：创建发货单、发货前下载 PDF、确认发货、下载 PDF、签收。
- 检查标准：发货前 PDF 返回 `400`；确认发货后库存扣减正确；PDF 返回 `application/pdf`；签收后状态为 `DELIVERED`。

### 15. 退货管理

- 测试方式：创建退货、处理退货、拒绝退货。
- 检查标准：处理退货后库存增加且状态为 `PROCESSED`；拒绝退货后状态为 `REJECTED`。

### 16. 成本统计

- 测试方式：创建成本记录、查询工单成本、读取成本统计。
- 检查标准：工单成本明细合计正确；统计接口返回数值。

### 17. 库存与统计

- 测试方式：查询产品库存、仪表盘、生产统计、质量统计。
- 检查标准：接口返回 `200`，结构可用于页面展示。

### 18. 清理校验

- 测试方式：清理自动化测试创建的 `TST-FULL-*` 夹具数据后再查残留；正式业务数据只允许归档。
- 检查标准：测试夹具不影响正式视图，正式数据未被物理删除。

## 建议执行顺序

1. 先做登录、权限、注册审核。
2. 再做物料、供应商、来料。
3. 然后做生产、领料、派工、报工、入库。
4. 最后做发货、退货、成本、统计。
5. 结束后执行清理校验。

## 备注

- 当前系统的已知完整实际测试基线为 `54` 项通过。
- 领料库存账的正确标准以本文件第 11 项为准。
