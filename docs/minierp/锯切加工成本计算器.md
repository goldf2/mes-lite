# 锯切加工成本计算器

## 1. 用途

根据整根材料的长度、重量和锯缝损耗，先估算单件材料成本，再结合人工工时、生产效率和机时费用测算单班收入，最后用多个产品的工时/机时组合测算更大规模跨度下的营业收入和经营利润。

## 2. 第一步：材料成本输入

- 材料长度：`mm`
- 材料总重量：`kg`
- 工件长度：`mm`
- 锯片厚度/锯缝：`mm`
- 原材料单价：`元/kg`
- 废屑回收单价：`元/kg`
- 剩余废料单价：`元/kg`
- 销售单价：`元/件`

## 3. 计算公式

```text
可加工数量 = floor((材料长度 + 锯片厚度) / (工件长度 + 锯片厚度))
线密度 = 材料总重量 / 材料长度
成品重量 = 可加工数量 × 工件长度 × 线密度
锯屑重量 = 可加工数量 × 锯片厚度 × 线密度
剩余废料重量 = 材料总重量 - 成品重量 - 锯屑重量
材料利用率 = 成品重量 / 材料总重量

净材料成本 = 材料总重量 × 原材料单价
               - 锯屑重量 × 废屑单价
               - 剩余废料重量 × 废料单价

单件净材料成本 = 净材料成本 / 可加工数量
单件材料毛利 = 销售单价 - 单件净材料成本
整根材料总毛利 = 可加工数量 × 销售单价 - 净材料成本
```

## 4. 第二步：班次收入

输入：

- 班次人数：`人`
- 每班时长：`小时`
- 人工小时成本：`元/小时`
- 生产效率：`件/人工时`
- 设备数量：`台`
- 机时成本：`元/小时`

公式：

```text
班次人工时 = 班次人数 × 每班时长
每班产量 = 班次人工时 × 生产效率
每班营业收入 = 每班产量 × 销售单价
每班材料成本 = 每班产量 × 单件净材料成本
每班人工成本 = 班次人工时 × 人工小时成本
每班机时费用 = 设备数量 × 每班时长 × 机时成本
每班经营利润 = 每班营业收入 - 每班材料成本 - 每班人工成本 - 每班机时费用
```

## 5. 第三步：混合产品规模测算

用户可录入多个产品，每个产品包含：

- 产品名称
- 计划数量
- 销售单价
- 单件材料成本
- 单件人工时
- 单件机时

同时输入计划班次数、每台每班机时和其他期间费用。

公式：

```text
规模营业收入 = Σ(产品数量 × 销售单价)
规模材料成本 = Σ(产品数量 × 单件材料成本)
规模人工时 = Σ(产品数量 × 单件人工时)
规模机时 = Σ(产品数量 × 单件机时)
规模人工成本 = 规模人工时 × 人工小时成本
规模机时费用 = 规模机时 × 机时成本
规模总成本 = 规模材料成本 + 规模人工成本 + 规模机时费用 + 其他期间费用
规模经营利润 = 规模营业收入 - 规模总成本
人工产能 = 班次人数 × 每班时长 × 计划班次数
机时产能 = 设备数量 × 每台每班机时 × 计划班次数
所需班次 = max(规模人工时 / 单班人工时, 规模机时 / 单班机时)
```

页面提供“同步当前产品”，用于把第一步和第二步算出的当前锯切产品参数写入混合产品行。

## 6. 边界

- 默认材料沿长度方向线密度均匀。
- 默认每件成品产生一次锯缝损耗。
- 废屑和剩余边料分别按各自回收单价冲减材料成本。
- 第一步只输出材料口径毛利。
- 第二步按单一产品和单班效率测算。
- 第三步按用户输入的产品组合测算规模营业收入，不自动读取产品工艺路线。
- 其他期间费用只在第三步整体扣除，不分摊到单件。

## 7. 后续扩展方向

后续根据可获得的真实数据，可逐步加入：

- 人工工时和单位工时费率。
- 设备折旧、占用时间和单位机时费率。
- 电力、冷却液等能源和辅料消耗。
- 锯片采购成本、寿命和单次切割分摊。
- 换料、换刀、调机、检验和包装成本。
- 废品率、返工率和订单批量影响。
- 管理费用、销售费用、税费和其他间接成本。

后续可把第三步的单件人工时、机时从产品工艺成本路线中自动带出。

## 8. 方案保存与对比

- 当前计算结果可按方案名称保存到系统。
- 第一步材料成本计算完成后即可保存为产品成本对象，不必完成班次收入和混合规模测算。
- 方案保存第一步材料输入、材料计算结果、第三步规模经营结果、单件人工时、单件机时和选中的加工工艺组合。
- 保存时可选择“临时产品”或绑定已有产品。临时产品不进入正式产品档案，但可作为后续混合测算的成本来源。
- 保存时可选择一个产品，把当前锯切测算作为 `SAWING_COST` 成本组成追加到该产品 BOM。
- 产品成本对象和 BOM 组成只保存单件材料成本、单件/千件人工工时、单件/千件机时和直接随产品发生的成本；房租、设备折旧等固定费用保留在经营测算中分摊，不写入 BOM 本体。
- 一个方案可关联锯切、钻孔、车削、铣削等多个工艺模板。
- 用户可勾选多个已保存方案，对比锯片厚度、可加工数量、材料利用率、单件材料成本、营业收入、总成本和经营利润率。
- 第三步可从已保存产品成本中调用条目，一键加入混合产品工时机时表。
- BOM 中的锯切成本组成只参与成本结构表达，不参与工单领料、库存预留和库存扣减。
- 当前复用 `sawing_cost_scenarios` 的生产成本字段保存规模测算结果：`labor_cost` 保存规模人工成本，`fixed_cost` 保存规模机时费用和其他期间费用，`full_cost/full_profit/full_margin` 保存规模总成本、经营利润和经营利润率。
