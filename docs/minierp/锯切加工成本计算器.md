# 锯切加工成本计算器

## 1. 用途

根据整根材料的长度、重量和锯缝损耗，估算可加工数量、材料利用率、单件净材料成本和材料毛利。

当前为第一阶段，只按材料、锯缝损耗和废料回收口径计算。后续将根据新增数据逐步扩展为更完整的加工利润模型。

## 2. 输入

- 材料长度：`mm`
- 材料总重量：`kg`
- 工件长度：`mm`
- 锯片厚度/锯缝：`mm`
- 原材料单价：`元/kg`
- 废屑回收单价：`元/kg`
- 剩余废料单价：`元/kg`
- 成品价格：`元/件`

## 3. 计算公式

```text
可加工数量 = floor((材料长度 + 锯片厚度) / (工件长度 + 锯片厚度))
线密度 = 材料总重量 / 材料长度
成品重量 = 可加工数量 × 工件长度 × 线密度
锯屑重量 = 可加工数量 × 锯片厚度 × 线密度
剩余废料重量 = 材料总重量 - 成品重量 - 锯屑重量
材料利用率 = 成品重量 / 材料总重量

净材料成本 = 材料总重量 × 原材料单价
               - 锯屑重量 × 废屑单价
               - 剩余废料重量 × 废料单价

单件净材料成本 = 净材料成本 / 可加工数量
单件材料毛利 = 成品价格 - 单件净材料成本
整根材料总毛利 = 可加工数量 × 成品价格 - 净材料成本
```

## 4. 边界

- 默认材料沿长度方向线密度均匀。
- 默认每件成品产生一次锯缝损耗。
- 废屑和剩余边料分别按各自回收单价冲减材料成本。
- 当前未计入人工、设备、能耗、包装、税费和管理费用，因此输出为“材料毛利”，不是完整净利润。

## 5. 后续扩展方向

后续根据可获得的真实数据，可逐步加入：

- 人工工时和单位工时费率。
- 设备折旧、占用时间和单位机时费率。
- 电力、冷却液等能源和辅料消耗。
- 锯片采购成本、寿命和单次切割分摊。
- 换料、换刀、调机、检验和包装成本。
- 废品率、返工率和订单批量影响。
- 管理费用、销售费用、税费和其他间接成本。

完整模型上线后，应同时保留第一阶段的快速估算模式，避免简单试算必须填写过多参数。
