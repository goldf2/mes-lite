# 锯切加工成本计算器

## 1. 用途

根据整根材料的长度、重量和锯缝损耗，估算可加工数量、材料利用率、单件净材料成本和材料毛利。

当前为第一阶段，只按材料、锯缝损耗和废料回收口径计算。后续将根据新增数据逐步扩展为更完整的加工利润模型。

## 2. 输入

- 材料长度：`mm`
- 材料总重量：`kg`
- 工件长度：`mm`
- 锯片厚度/锯缝：`mm`
- 原材料单价：`元/kg`
- 废屑回收单价：`元/kg`
- 剩余废料单价：`元/kg`
- 成品价格：`元/件`

## 3. 计算公式

```text
可加工数量 = floor((材料长度 + 锯片厚度) / (工件长度 + 锯片厚度))
线密度 = 材料总重量 / 材料长度
成品重量 = 可加工数量 × 工件长度 × 线密度
锯屑重量 = 可加工数量 × 锯片厚度 × 线密度
剩余废料重量 = 材料总重量 - 成品重量 - 锯屑重量
材料利用率 = 成品重量 / 材料总重量

净材料成本 = 材料总重量 × 原材料单价
               - 锯屑重量 × 废屑单价
               - 剩余废料重量 × 废料单价

单件净材料成本 = 净材料成本 / 可加工数量
单件材料毛利 = 成品价格 - 单件净材料成本
整根材料总毛利 = 可加工数量 × 成品价格 - 净材料成本
```

## 4. 边界

- 默认材料沿长度方向线密度均匀。
- 默认每件成品产生一次锯缝损耗。
- 废屑和剩余边料分别按各自回收单价冲减材料成本。
- 当前未计入人工、设备、能耗、包装、税费和管理费用，因此输出为“材料毛利”，不是完整净利润。

## 5. 后续扩展方向

后续根据可获得的真实数据，可逐步加入：

- 人工工时和单位工时费率。
- 设备折旧、占用时间和单位机时费率。
- 电力、冷却液等能源和辅料消耗。
- 锯片采购成本、寿命和单次切割分摊。
- 换料、换刀、调机、检验和包装成本。
- 废品率、返工率和订单批量影响。
- 管理费用、销售费用、税费和其他间接成本。

完整模型上线后，应同时保留第一阶段的快速估算模式，避免简单试算必须填写过多参数。

## 6. 方案保存与对比

- 当前计算结果可按方案名称保存到系统。
- 方案同时保存全部输入、计算结果和选中的加工工艺组合。
- 一个方案可关联锯切、钻孔、车削、铣削等多个工艺模板。
- 用户可勾选多个已保存方案，对比锯片厚度、可加工数量、材料利用率、单件成本、材料毛利和毛利率。
- 第一阶段的“工艺组合”是方案标识和对比维度，除锯切参数外，其他工艺尚未自动产生额外成本。

## 7. 三阶段完整成本模型

### 基础费用

- 净材料成本由锯切模型自动计算。
- 用户可增加耗材、能源、外协、包装等直接费用。
- 支持直接金额和“数量 × 单价”。
- 可将废料回收等项目设置为抵扣。

### 人员费用

- 支持“人数 × 每人工时 × 小时费率”。
- 支持“计件数量 × 计件单价”。
- 支持直接录入人员费用总额。

### 固定费用

- 支持厂房租金、设备折旧、管理费用等项目。
- 周转分摊公式：`周期费用 ÷ 周期周转量 × 本方案占用量`。
- 也支持直接录入本方案应分摊的固定金额。

页面同时输出：

```text
基础直接成本及直接利润率
基础费用 + 人员费用 = 制造成本及制造利润率
制造成本 + 固定费用 = 完整生产成本及完整利润率
```
